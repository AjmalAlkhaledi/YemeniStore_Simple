# YemeniStore — متجر إلكتروني (المحاضرة السادسة)

تطبيق متجر إلكتروني صغير مبني بـ Flutter باستخدام **Provider** لإدارة الحالة.

## الميزات
- الصفحة الرئيسية: عرض المنتجات مقسّمة حسب الأقسام (إلكترونيات، ملابس، كتب، عطور، مستلزمات المطبخ).
- شاشة الفئات: قائمة بالأقسام، والضغط على قسم ينتقل لشاشة منتجات هذا القسم فقط.
- شاشة تفاصيل المنتج: صورة، سعر، وصف، وزرّا «إضافة للسلة» و«إضافة للمفضلة».
- شاشة السلة: المنتجات المضافة مع تعديل الكميات والإجمالي.
- شاشة المفضلة: المنتجات المضافة للمفضلة.
- عدّاد السلة يتحدّث في كل مكان عبر Provider.

## التشغيل محلياً
```bash
flutter pub get
flutter run
```

## النشر على الويب (Netlify)
```bash
flutter build web
```
ثم اسحب مجلد `build/web` وأفلته على موقع Netlify للحصول على رابط مباشر.
